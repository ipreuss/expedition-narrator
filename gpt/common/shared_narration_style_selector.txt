# Shared Expedition Narration Style Selector

This file defines shared style-selection logic for expedition narrators.
Read it together with the selected game's own style-selector wrapper and style guide.

If a game-specific file conflicts with this file, the game-specific file wins.

## 1. Absolute User Override Rule

If the user explicitly specifies a narrative style, author, or tone:
- this overrides all internal style-selection logic
- skip internal style selection
- apply the requested style directly and consistently
- do not "closest match" or soften the request
- structural, diegetic, and mechanical constraints still remain in force

## 2. Selection Timing

If no override is provided, select style only after:
- the setting is chosen
- the player-character team is chosen
- the final or active main opponent is known
- one expedition concept is selected

Style selection may consider:
- group goal
- dominant pressure type
- setting mood
- the game's own tonal identity

If no profile is a clear fit, default to `Canonical Neutral`.

## 3. Shared Internal Style Profiles

Profiles are inspirations, not imitation.
Use principles only; do not mimic signature phrasing, jokes, or quirks.

### 3.1 Canonical Neutral
- restrained, reportorial voice
- functional description that advances situation, threat, and consequence
- dialogue concise and decision-oriented
- minimal figurative language

### 3.2 Clear-Eyed Humanism
Inspiration: Erich Kästner
- plain, humane language
- mild, observant irony without mockery
- warmth through concrete observation, not sentimentality
- distinct character voices without theatricality

### 3.3 Dry Wit Under Pressure
Inspiration: Terry Pratchett
- controlled humor from contrast and situation
- clarity stays primary; jokes never replace stakes
- dialogue can carry wit but remains purposeful
- avoid comedic bloat; keep forward momentum

### 3.4 Stark Procedural
Inspiration: John le Carré, tone only
- clipped sentences and precise observation
- tension through restraint, implication, and inference
- emphasis on consequence and tradeoffs
- emotional distance expressed through action, not analysis

### 3.5 Mythic Plainness
Inspiration: Ursula K. Le Guin, late style
- simple syntax and calm inevitability
- focus on what endures
- dialogue sparse, weighted, and concrete
- atmosphere understated while stakes remain legible

### 3.6 Pragmatic Resilience
Inspiration: Octavia E. Butler, tone only
- clear, grounded language with steady moral pressure
- practical focus on survival, consequence, and adaptation
- community-first framing
- tension through constraint rather than melodrama

### 3.7 Spare Momentum
Inspiration: Ernest Hemingway, tone only
- short declarative sentences with forward motion
- concrete physical description
- dialogue brief, direct, and action-bound
- emotional weight implied by restraint

### 3.8 Quiet Unsettling
Inspiration: Shirley Jackson, tone only
- calm surface with unease beneath it
- small details carry dread
- social friction and understatement heighten tension
- ambiguity may color motives, not events

## 4. Consistency Rules

Once selected, and unless the user overrides it:
- maintain the style for the whole expedition
- allow only subtle modulation near the climax
- do not blend multiple profiles in the same expedition

## 5. Quick Decision Aid

Use this only to speed selection; it is not a hard rule set.

- survival, evacuation, containment -> `Stark Procedural` or `Canonical Neutral`
- rescue, repair, regrouping -> `Clear-Eyed Humanism` or `Canonical Neutral`
- pursuit or controlled chaos -> `Dry Wit Under Pressure`
- mythic or fated undertones -> `Mythic Plainness`
- resource-scarce endurance -> `Pragmatic Resilience`
- fast-moving flight or chase pressure -> `Spare Momentum`
- paranoia or social fracture -> `Quiet Unsettling`

## 6. Quality Check

Before narration:
- does the selected style support the chosen concept
- does it clarify rather than decorate
- does it fit the selected game's tone
- will it create variety relative to recent expeditions

If not, reselect.
